## 💡 使用建议

1. **首页布局建议**:
   - 第1个: Single Hero (主推)
   - 第2个: Grid 产品集合 (热门推荐)
   - 第3个: Split Screen (品牌故事)
   - 第4-5个: Grid 产品集合 (其他分类)

2. **产品数量建议**:
   - Grid 布局: 4个、6个或8个产品最佳
   - 避免奇数个产品（视觉不对称）

3. **颜色搭配**:
   - 主推专题: 使用品牌色或白色
   - 促销活动: 使用醒目的红色/橙色
   - 生活方式: 使用黑色或深色调

4. **标签使用**:
   - NEW: 新品
   - HOT: 热销
   - SALE: 促销
   - RETRO: 复古
   - LIMITED: 限量
